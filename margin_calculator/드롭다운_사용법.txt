# 📋 드롭다운 방식 마진 계산기 사용법

## 🎯 드롭다운이 적용된 열들

| 열 | 항목명 | 드롭다운 옵션 | 설명 |
|:---|:---|:---|:---|
| B | 판매 방식 | 직접 배송 / 풀필먼트 | 배송 방식 선택 |
| D | 매입가 부가세 | 포함 / 미포함 | 매입가의 부가세 여부 |
| F | 판매가 부가세 | 포함 / 미포함 | 판매가의 부가세 여부 |
| G | 플랫폼 | 설정 시트의 플랫폼 목록 | 판매 플랫폼 선택 |

## 📝 사용 방법

### 1. **B열 (판매 방식) 선택**
1. B열 셀 클릭
2. 드롭다운 화살표(▼) 클릭  
3. **"직접 배송"** 또는 **"풀필먼트"** 선택
4. 선택에 따라 비용 계산이 자동 변경됩니다

### 2. **D열, F열 (부가세 여부) 선택**
1. 해당 셀 클릭
2. 드롭다운에서 **"포함"** 또는 **"미포함"** 선택
3. 부가세 계산이 자동으로 적용됩니다

### 3. **G열 (플랫폼) 선택**
1. G열 셀 클릭
2. 드롭다운에서 원하는 플랫폼 선택
3. 자동으로 연동되는 항목들:
   - H열: 플랫폼 수수료율 자동 입력
   - K열: 풀필먼트 수수료 기본값 자동 입력

## ⚡ 자동화 기능

### **판매 방식에 따른 비용 계산**
- **직접 배송**: 매입가 + 배송비 + 포장비 + 기타비용
- **풀필먼트**: 매입가 + 풀필먼트 수수료 + 기타비용

### **플랫폼 연동 기능**
- 플랫폼 선택 → 수수료율 자동 적용
- 플랫폼 선택 → 기본 풀필먼트 비용 자동 입력
- 필요시 풀필먼트 비용 수동 조정 가능

## 💡 사용 팁

1. **드롭다운 우선**: 먼저 드롭다운 항목들을 선택하세요
2. **플랫폼 먼저**: 플랫폼을 선택하면 관련 수수료들이 자동 입력됩니다
3. **수정 가능**: 자동 입력된 값들도 필요시 수정할 수 있습니다
4. **실시간 계산**: 값 변경 시 즉시 마진이 재계산됩니다

## 🔄 입력 순서 (권장)

1. **상품명** 입력 (A열)
2. **판매 방식** 드롭다운 선택 (B열)
3. **매입가 & 부가세 여부** 입력 (C, D열)
4. **판매가 & 부가세 여부** 입력 (E, F열)
5. **플랫폼** 드롭다운 선택 (G열) → 수수료 자동 입력
6. 필요시 **배송비, 포장비, 기타비용** 입력
7. **마진 분석 결과** 자동 확인!

이제 클릭만으로 간편하게 선택할 수 있습니다! 🎉
