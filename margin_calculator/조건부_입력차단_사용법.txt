# 🚫 조건부 입력 차단 기능 사용법

## ✅ **완성된 기능**

### **1. 풀필먼트 선택 시**
- ❌ **I열 (배송비)**: 입력 차단 + 회색 배경
- ❌ **J열 (포장비)**: 입력 차단 + 회색 배경  
- ✅ **K열 (풀필먼트 수수료)**: 자동 입력

### **2. 직접 배송 선택 시**
- ✅ **I열 (배송비)**: 입력 가능
- ✅ **J열 (포장비)**: 입력 가능
- ❌ **K열 (풀필먼트 수수료)**: 회색 배경 (자동 입력)

## 🎯 **작동 방식**

### **입력 차단**
1. 풀필먼트 선택 후 배송비/포장비 입력 시도
2. **에러 메시지 표시**: "풀필먼트 방식에서는 배송비를 입력할 수 없습니다"
3. 입력이 차단되고 셀이 비워짐

### **시각적 표시**
- 🔘 **활성 상태**: 일반 배경 (흰색)
- 🚫 **비활성 상태**: 회색 배경 (입력 불가)
- 📝 **헤더 도움말**: 
  - "배송비 (직접배송만)"
  - "포장비 (직접배송만)"  
  - "풀필먼트 수수료 (자동입력)"

## 📋 **사용 단계**

1. **B열에서 판매 방식 선택**
   - 직접 배송 → I, J열 활성화
   - 풀필먼트 → I, J열 비활성화 (회색)

2. **해당하는 비용만 입력**
   - 직접 배송: 배송비, 포장비 입력
   - 풀필먼트: 풀필먼트 수수료는 자동 입력됨

3. **잘못 입력 시 자동 차단**
   - 에러 메시지와 함께 입력 차단
   - 올바른 판매 방식 선택 후 재입력

## 💡 **장점**

- **실수 방지**: 잘못된 조합 입력 불가
- **시각적 안내**: 회색 배경으로 비활성 상태 표시
- **사용자 친화적**: 명확한 에러 메시지 제공
- **자동화**: 판매 방식 변경 시 즉시 반영
