# 마진 분석 계산기 - 데이터 유효성 검사 설정 방법

Excel 파일에서 드롭다운 메뉴를 설정하는 방법입니다.

## 1. B열 (판매 방식) 드롭다운 설정

1. B2 셀을 선택합니다
2. 데이터 탭 > 데이터 유효성 검사 클릭
3. 허용: 목록 선택
4. 원본: "직접 배송,풀필먼트" 입력
5. 확인 클릭
6. B2 셀을 복사하고 B3:B1000 범위에 붙여넣기

## 2. D열 (매입가 부가세) 드롭다운 설정

1. D2 셀을 선택합니다
2. 데이터 탭 > 데이터 유효성 검사 클릭
3. 허용: 목록 선택
4. 원본: "포함,미포함" 입력
5. 확인 클릭
6. D2 셀을 복사하고 D3:D1000 범위에 붙여넣기

## 3. F열 (판매가 부가세) 드롭다운 설정

1. F2 셀을 선택합니다
2. 데이터 탭 > 데이터 유효성 검사 클릭
3. 허용: 목록 선택
4. 원본: "포함,미포함" 입력
5. 확인 클릭
6. F2 셀을 복사하고 F3:F1000 범위에 붙여넣기

## 4. G열 (플랫폼) 드롭다운 설정

1. G2 셀을 선택합니다
2. 데이터 탭 > 데이터 유효성 검사 클릭
3. 허용: 목록 선택
4. 원본: =설정_각종_수수료!$A$2:$A$9 입력
5. 확인 클릭
6. G2 셀을 복사하고 G3:G1000 범위에 붙여넣기

## 추가 플랫폼 등록 방법

'설정_각종_수수료' 시트에서:
1. A열에 새 플랫폼명 입력
2. B열에 해당 플랫폼의 수수료율(%) 입력
3. G열 드롭다운의 원본 범위를 새 데이터까지 확장

## 주의사항

- 수식이 올바르게 작동하려면 플랫폼명이 정확히 일치해야 합니다
- 마진율(%)은 자동으로 퍼센트 형식으로 표시됩니다
- 판매 방식에 따라 총 비용 계산이 자동으로 달라집니다
