# 마진 분석 계산기 - 토글 기능 사용법

## 🔄 스마트 토글 기능

### 1. 판매 방식 토글 (B열)
- **직접 배송**: I열(배송비), J열(포장비) 사용
- **풀필먼트**: K열(풀필먼트 수수료) 사용
- 선택에 따라 총 비용 계산이 자동으로 변경됩니다

### 2. 플랫폼 스마트 선택 (G열)
- 플랫폼을 선택하면:
  - H열(플랫폼 수수료%) 자동 입력
  - K열(풀필먼트 수수료) 기본값 자동 입력
- 원하시면 K열 값을 수정 가능합니다

### 3. 부가세 토글 (D열, F열)
- **포함**: 부가세가 이미 포함된 가격
- **미포함**: 부가세를 별도로 계산할 가격

## 📋 간편 사용 단계

1. **상품명** 입력 (A열)
2. **판매 방식** 선택 → 관련 비용 항목 활성화
3. **매입가** & **부가세 여부** 입력
4. **판매가** & **부가세 여부** 입력
5. **플랫폼** 선택 → 수수료 & 풀필먼트 비용 자동 입력
6. 필요시 **배송비, 포장비, 기타 비용** 입력
7. **모든 마진 정보 자동 계산** 완료!

## 💡 토글의 장점

- **원클릭 선택**: 드롭다운에서 한 번만 선택
- **자동 연동**: 관련 항목들이 자동으로 설정
- **실시간 계산**: 값 변경 시 즉시 마진 재계산
- **오류 방지**: 잘못된 조합 방지

## 🔧 커스터마이징

**설정_각종_수수료** 시트에서:
- 새 플랫폼 추가
