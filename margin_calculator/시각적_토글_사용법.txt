# 🔘 시각적 토글 버튼 사용법

## 📋 토글 버튼이 있는 새로운 열 구성

| 열 | 항목 | 설명 |
|:---|:---|:---|
| A | 상품명 | 상품 이름 입력 |
| B | 직접배송 | 🔘 토글 버튼 (○ 클릭 시 ● 로 변경) |
| C | 풀필먼트 | 🔘 토글 버튼 (○ 클릭 시 ● 로 변경) |
| D | 매입가 | 원가 입력 |
| E | 매입가 부가세 | 포함/미포함 선택 |
| F | 판매가 | 판매 가격 입력 |
| G | 판매가 부가세 | 포함/미포함 선택 |
| H | 플랫폼 | 플랫폼 선택 |
| I | 플랫폼 수수료(%) | 자동 계산 |
| J | 배송비 | (직접배송 선택시) |
| K | 포장비 | (직접배송 선택시) |
| L | 풀필먼트 수수료 | 자동 입력 (수정 가능) |
| M | 기타 비용 | 추가 비용 입력 |

## 🎯 토글 버튼 사용 방법

### 1. 직접배송 선택하기
1. **B열(직접배송)** 의 ○ 를 클릭
2. 수동으로 **●** 로 변경
3. C열(풀필먼트)은 **○** 로 유지
4. J열(배송비), K열(포장비) 입력 활성화

### 2. 풀필먼트 선택하기  
1. **C열(풀필먼트)** 의 ○ 를 클릭
2. 수동으로 **●** 로 변경
3. B열(직접배송)은 **○** 로 유지
4. L열(풀필먼트 수수료) 자동 입력

### 3. 토글 전환하기
- 한쪽을 **●** 로 하면 다른쪽은 **○** 로 설정
- 한 번에 하나만 ● 상태로 유지

## ⚡ 자동화 기능

### 플랫폼 선택 시
- H열에서 플랫폼 선택 → I열 수수료율 자동 입력
- L열 풀필먼트 수수료도 기본값 자동 입력

### 비용 계산
- **직접배송 (B열이 ●)**: 매입가 + 배송비 + 포장비 + 기타비용
- **풀필먼트 (C열이 ●)**: 매입가 + 풀필먼트수수료 + 기타비용

## 💡 사용 팁

1. **토글은 수동 변경**: ○ 를 ● 로 직접 타이핑
2. **하나만 선택**: 직접배송과 풀필먼트 중 하나만 ●
3. **시각적 확인**: ● 표시된 방식으로 비용이 계산됨
4. **플랫폼 우선**: 플랫폼 선택 후 다른 값들이 자동 입력

## 🎨 토글 버튼 디자인

- **○**: 비활성화 (회색 배경)
- **●**: 활성화 (녹색 배경으로 설정됨)
- **테두리**: 명확한 구분을 위한 경계선

이제 클릭만으로 판매 방식을 쉽게 선택할 수 있습니다!
